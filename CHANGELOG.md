## [1.0.0] - 2025-05-30

### Added
- Initial release of Free API Collection
- 100+ curated APIs across 12 categories
- AI & Machine Learning APIs (Groq, Gemini, HuggingFace, etc.)
- Security & OSINT APIs (Shodan, VirusTotal, HaveIBeenPwned, etc.)
- Weather & Environment APIs (OpenWeather, Open-Meteo, etc.)
- Finance & Crypto APIs (CoinGecko, Alpha Vantage, Binance, etc.)
- Web & Internet APIs (IPify, IP-API, URLScan, etc.)
- Social Media APIs (GitHub, Reddit, Telegram, Discord, etc.)
- Maps & Location APIs (OpenStreetMap, Mapbox, etc.)
- Entertainment APIs (TMDB, Spotify, RAWG, etc.)
- News & Media APIs (NewsAPI, GNews, etc.)
- Communication APIs (Telegram Bot, SendGrid, Twilio, etc.)
- Developer Tools APIs (JSONPlaceholder, ReqRes, etc.)
- Miscellaneous APIs
